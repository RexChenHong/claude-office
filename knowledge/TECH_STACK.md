# Claude Office - 技術知識庫

## 技術選型分析

### 🎮 遊戲引擎/框架

#### PixiJS ⭐ 主要候選
- **類型**：2D WebGL 渲染引擎
- **優點**：
  - 輕量、效能好
  - 適合 2D 場景
  - 豐富的動畫支援
  - 活躍的社群
- **缺點**：
  - 需要額外處理角色動畫系統
  - 沒有內建狀態機
- **適合度**：★★★★☆

#### Phaser
- **類型**：2D 遊戲框架
- **優點**：
  - 完整的遊戲系統（狀態、物理、動畫）
  - 內建 Tween 動畫
  - 適合 NPC 行為
- **缺點**：
  - 比 PixiJS 重一些
- **適合度**：★★★★☆

#### PixelLab API
- **類型**：像素藝術 API
- **優點**：
  - 可程式化生成像素風格
- **缺點**：
  - 不是遊戲引擎
  - 需要配合其他框架
- **適合度**：★★★☆☆（作為輔助工具）

#### Three.js
- **類型**：3D WebGL 引擎
- **優點**：
  - 可做 3D 場景
- **缺點**：
  - 對這個專案來說可能太重
  - 3D 資產製作成本高
- **適合度**：★★☆☆☆

#### AI 圖像生成
- **Stable Diffusion**：可生成原神風格角色
- **Midjourney**：風格一致性好
- **ComfyUI**：可本地運行，控制精細

#### 動畫工具
- **Live2D**：日系角色動畫標準
- **Spine**：2D 骨骼動畫
- **DragonBones**：免費骨骼動畫工具
