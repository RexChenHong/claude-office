# Claude Office - 角色設計

## 概述
5 名女性角色，日系原神風格，各有獨特個性與外觀。

---

## 角色列表

### 1. 🌸 櫻 (Sakura)
**原型：辦公室領袖 / 優雅秘書長**

| 屬性 | 描述 |
|-----|------|
| **髮色** | 櫻花粉長髮 |
| **瞳色** | 淡紫色 |
| **服裝** | 白色襯衫 + 粉色西裝外套 + 黑色鉛筆裙 |
| **配件** | 眼鏡、珍珠耳環 |
| **性格** | 優雅、可靠、有領導氣質、偶爾嚴肅 |
| **閒置動作** | 喝咖啡、整理文件、看手錶 |
| **辦公桌風格** | 井然有序、小盆栽、精緻筆筒 |

---

### 2. 🔥 焰 (Homura)
**原型：行動派 / 熱血工程師**

| 屬性 | 描述 |
|-----|------|
| **髮色** | 火紅色短髮（馬尾） |
| **瞳色** | 琥珀色 |
| **服裝** | 黑色 T 恤 + 牛仔外套 + 休閒褲 |
| **配件** | 耳機掛脖、運動手錶 |
| **性格** | 熱血、直接、偶爾衝動、專注時會嘟嘴 |
| **閒置動作** | 伸懶腰、轉筆、踢腳 |
| **辦公桌風格** | 雙螢幕、機械鍵盤、一堆能量飲料罐 |

---

### 3. 💧 涼 (Ryo)
