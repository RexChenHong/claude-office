## 🐛 已知問題

### VPS WebGL 問題
- **現象**：VPS 上的 Chrome Relay 無法創建 WebGL 上下文
- **原因**：NVIDIA GPU + Mesa llvmpipe 衝突
- **解決方案**：
  1. 使用本地瀏覽器測試 ✅
  2. 更新 VPS GPU 驅動（需要系統權限）
