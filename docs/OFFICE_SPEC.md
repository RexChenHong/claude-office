## 🏗️ 建模規格

### A. 地板與牆面

#### 地板
- **材質**：木質地板或灰色地磚
- **顏色**：淺灰色（#E8E8E8）或淺木色（#D4B896）
- **紋理**：
  - 木地板：長條狀拼接（1200mm x 120mm）
  - 地磚：方形拼接（600mm x 600mm）
- **反光**：輕微反光（Roughness: 0.7）

#### 牆面
- **材質**：油漆牆面
- **顏色**：純白色（#FFFFFF）或淺灰色（#F5F5F5）
- **高度**：3 米（標準層高）
- **細節**：
  - 踢腳板（高度：100mm，顏色：白色或灰色）
  - 電源插座（高度：300mm）

#### 1. 辦公桌（Workstation Desk）

## 🎯 Blender 建模步驟

### Phase 1：基礎結構（2-3 小時）
1. 創建地板平面（20m x 20m）
2. 添加牆面（高度 3m）
3. 創建窗戶開口
4. 添加踢腳板

### Phase 2：辦公家具（3-4 小時）
1. 建模辦公桌（Box + Bevel）
2. 建模辦公椅（基礎形狀）
3. 添加電腦設備
4. 添加辦公用品

### Phase 3：休息區域（2-3 小時）
1. 建模沙發
2. 建模茶几
3. 添加地毯
4. 添加裝飾品

### Phase 4：材質與燈光（2-3 小時）
1. 設置 PBR 材質
2. 配置燈光系統
3. 添加陰影
4. 渲染測試

### Phase 5：優化與導出（1-2 小時）
1. 降低多邊形數量
2. UV 展開
3. 導出 GLTF 格式
4. Three.js 整合測試
