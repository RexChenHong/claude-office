## 📏 詳細配置

### 1. 會議室（左上角）
- **尺寸**：3m x 4m
- **設備**：
  - 橢圓形會議桌（1.5m x 1m）
  - 6 張會議椅
  - 白板（2m x 1.2m）
  - 玻璃隔間（透明）

### 2. 工作區域（中央）
- **配置**：5 個獨立工作站（一字排開）
- **間距**：每個工作站 2m 間隔
- **設備**：
  - 辦公桌（1.2m x 0.6m）
  - 人體工學椅
  - 電腦螢幕（雙螢幕）
  - 辦公用品

### 3. 公共區域（右下角）
- **休息區**：
  - L型沙發
  - 茶几
  - 地毯
- **咖啡區**：
  - 咖啡桌
  - 高腳椅（2張）
  - 咖啡機

## 🎨 視覺改進

### 材質升級
1. **環境貼圖**：HDR 環境貼圖（城市景觀）
2. **紋理貼圖**：
   - 木地板紋理
   - 布料紋理（沙發、椅子）
   - 金屬紋理（框架）
3. **PBR 材質**：物理正確的材質參數

### 燈光優化
1. **環境光**：半球光 + 環境貼圖
2. **主光源**：模擬自然光（窗戶）
3. **室內照明**：LED 平板燈（天花板）
4. **裝飾燈**：檯燈、落地燈

### 後處理增強
1. **SSAO**：環境光遮蔽
2. **Bloom**：泛光效果（增強）
3. **景深**：DOF 效果
4. **色彩校正**：LUT 色彩查找表

## 🏗️ 建模細節

### 隔間設計
- **玻璃隔間**：透明玻璃 + 金屬框架
- **半開放式**：保持通透感
- **隔音**：玻璃材質（厚度 10mm）

### 窗戶設計
- **落地窗**：大面積窗戶
- **窗簾**：白色紗簾
- **採光**：自然光充足

### 天花板
- **高度**：2.8m
- **燈具**：LED 平板燈（600mm x 600mm）
- **通風口**：空調出風口
