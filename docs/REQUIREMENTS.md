# Claude Office - 需求文檔

## 專案概述
一個視覺化辦公室場景，以遊戲/動畫方式展示 Claude session 的即時狀態。

## 核心需求

### 1. 場景設計
- **辦公室佈景**：分為工作區和沙發休息區
- **工作區**：5 個辦公桌位置
- **休息區**：5 個座位（沙發/椅子）

### 2. 角色設計
- **數量**：5 名常駐角色（統一女性）
- **風格**：日系原神畫風
- **每個角色有專屬辦公桌位置**

### 3. 角色行為狀態

| Session 狀態 | 角色行為 |
|-------------|---------|
| 無 Session | 在休息區閒置（idle 動作） |
| Session 開啟 | 從休息區起身 → 走到辦公桌 → 坐下 |
| Session 工作（輸出中） | 打字動作 |
| Session 等待中 | 停下來滑手機 |
| Session 關閉 | 站起來 → 走回休息區 → 坐下 |

### 4. 技術要求
- **框架**：PixiJS（確定）
- **畫風**：日系原神風格（確定，直接衝）
- **遊戲概念**：場景 + NPC 系統
- **硬體限制**：GPU+VRAM 總使用率 ≤ 10%（其他資源留給主專案模型訓練）
- **部署目標**：網頁，透過 `http://100.113.156.108:8051/` 存取（Chrome 瀏覽器）

### 5. Session 監聽需求
- **不能影響 session 本身運作**
- **只需四個狀態**：
  - 開啟（session 建立時）
  - 工作（session 正在輸出/處理）
  - 閒置（session 等待用戶輸入）
  - 關閉（session 結束）

### 6. 角色設計
- **5 名女性角色，各有個性差異**（交由天愛星規劃）

### 5. 工作資料夾規範
- **路徑**：`/mnt/e_drive/claude-office`
- **必須包含**：
  - 進度紀錄
  - 知識譜
  - 腳手架
  - 需求文檔

### 6. 整合需求
- **監控目標**：Claude CLI 的 session（不是 OpenClaw）
- 需要與 Claude CLI session 狀態連動
- 即時反映 session 變化

### 7. 版本控管
- **GitHub Repo**：https://github.com/RexChenHong/claude-office.git
- 使用 git worktree 概念管理

### 8. 美術規格
- **角色比例**：正常比例（6 頭身），接近原神原圖
- **來源**：AI 生成（Stable Diffusion）

## 設計方向
- 日系動漫風格（參考原神）
- 辦公室溫馨感
- 角色有個性差異（可選）
